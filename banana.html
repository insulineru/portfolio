Лендинг про банан.

Данный проект стал первым практическим проектом, не считая учебных, который я решил сделать. Дизайн был разработан различными изменениями и дополнениями данного материала.

Поскольку многие компоненты были внешне похожи с стандартными компонентами Bootstrap, то и для разработки был выбран данный фреймворк.
Проект адаптивен для мобильного и настольного разрешения, но имеются некоторые нюансы, которые мешают корректному отображению.

В работе применялись технологии:

  • Node.JS - для сборки проекта и запуска остальных библиотек.
  • Yarn - для более быстрой установки модулей, чем в NPM.
  • SASS - для написания стилей сайта.
  • PostCSS, Autoprefixer - для добавления сайту идентичности при работе в разных браузерах.
  • Bootstrap - как основной фреймворк проекта, предусматривающий стилизацию кнопок, ссылок, меню, секций, а так же сетки.
  • Sketch - для работы с дизайном.
  • OwlCarousel, jQuery - для создания анимированных слайдеров и динамического Range поля.
