Uber Eats

Макет проекта был взял у Евгения Родионова на его базовом интенсиве по вёрстке.

В описании проекта было предложено делать его с помощью Bootstrap Grid, но в данном случае я решил сделать сетку без использования сторонних модулей.
Проект адаптивен для мобильного, планшетного и настольного разрешения.

В работе применялись технологии:

  • Node.JS - для сборки проекта и запуска остальных библиотек.
  • Yarn - для более быстрой установки модулей, чем в NPM.
  • БЭМ - для наименования классов.
  • SCSS(Sass) - для написания стилей сайта.
  • PostCSS, Autoprefixer - для добавления сайту идентичности при работе в разных браузерах.
  • Editorconfig, Stylelint - для проверки написанного кода на соответствие стайлгайдам.
  • Figma - для работы с дизайн-макетом.
